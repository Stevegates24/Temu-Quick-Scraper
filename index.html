Features

Built to move
as fast as you do.

No buttons to hunt. No popups to dismiss. Open the extension and it's already done.

⚡
Auto-scrapes on open
The moment you click the icon, scraping begins. By the time you look at it, your clipboard is already loaded.
📌
Persists between sessions
Close the popup, switch tabs, paste in your tool — data is still there when you come back. Clears only on a new product.
📋
Clipboard-first output
All 5 fields copied as tab-separated text on every scrape. Paste in one shot. Click fields individually for single values.
🔗
Clean ATC URL
Builds the Add-to-Cart URL with goods_id and sku_id from the page — no cart, no address bar fumbling.
🌍
All Temu regions
Works on .com, .de and every other regional Temu site. Same page structure, same scraper, everywhere.
🛡️
Zero data collected
Runs entirely in your browser. No servers, no accounts, no analytics. Your research stays yours.
How it works

Three steps.
That's it.

1
Install the extension
Download the zip, unzip it, open your browser's extension page and load unpacked. Under 60 seconds, start to finish.
chrome://extensions → Load unpacked
2
Open any Temu product page
Navigate to a product on temu.com, let the page load, then click the extension icon in your toolbar.
3
Paste — you're done
All 5 fields are already in your clipboard. Click into your tool and paste. Or open the popup to copy one field at a time.
Ctrl+V → 5 fields, instantly

Install

Pick your browser.

Available on Chrome and Firefox — same extension, both supported.

🌐
Google Chrome
Download the zip and load unpacked via chrome://extensions. Developer mode, one click.
→
🦊
Firefox
Install directly from Firefox Add-ons in one click — no developer mode needed.
→

Manual install (Chrome developer mode)

  1. Download the .zip from GitHub and unzip it
  2. Go to chrome://extensions and enable Developer Mode
  3. Click Load unpacked and select the unzipped folder
  4. Open any Temu product page and click the extension icon
